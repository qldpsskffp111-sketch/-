# 🎨 나만의 전남 여행 지도 (Jeonnam Travel Map)

전라남도의 주요 관광지를 한눈에 확인하고, 나만의 방문 기록을 남길 수 있는 인터랙티브 웹 지도 서비스입니다.

## 🌟 주요 기능

- **인터랙티브 지도**: 전라남도 22개 시·군의 위치와 대표 관광지를 확인할 수 있습니다.
- **방문 기록 관리**: 각 지역별 방문 횟수를 체크하고, 방문한 지역은 지도상에 색상으로 표시됩니다.
- **이미지 저장**: 내가 만든 여행 지도를 PNG 이미지 파일로 저장할 수 있습니다. (html2canvas 활용)
- **공유하기**: 나만의 방문 기록이 담긴 지도를 고유한 링크로 생성하여 친구에게 공유할 수 있습니다.
- **데이터 보존**: 브라우저의 로컬 스토리지(LocalStorage)를 사용하여 새로고침 후에도 기록이 유지됩니다.

## 🛠 사용 기술

- **Language**: HTML5, CSS3, JavaScript (ES6+)
- **Visualization**: SVG (Scalable Vector Graphics)
- **Library**: [html2canvas](https://html2canvas.hertzen.com/) (이미지 캡처 기능)

## 🚀 시작하기

1. 이 저장소를 클론하거나 HTML 파일을 다운로드합니다.
2. `jeonnam_travel_map (1).html` 파일을 브라우저에서 엽니다.
3. 지도의 지역을 클릭하거나 `+` 버튼을 눌러 여행 기록을 시작하세요!
